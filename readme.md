# Scientific Calculator

A fully functional scientific calculator that handles basic arithmetic and advanced scientific operations like trigonometric functions, logarithms, exponents, and more. Built with a clean, responsive UI for a smooth user experience across all devices.

## 🚀 Demo

Try it live: [https://scientific-calculator-wheat.vercel.app/](https://scientific-calculator-wheat.vercel.app/)

## 🔧 Features

- Basic operations: addition, subtraction, multiplication, division
- Advanced functions: sin, cos, tan, log, ln, square root, exponents
- Clear and Delete functionality
- Responsive design (works on mobile, tablet, desktop)
